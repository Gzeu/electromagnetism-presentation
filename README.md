## 📚 Conținut Educațional Detaliat

### Ecuațiile lui Maxwell

1. **∇ · E = ρ/ε₀** - Legea lui Gauss pentru câmpul electric
2. **∇ · B = 0** - Legea lui Gauss pentru câmpul magnetic
3. **∇ × E = -∂B/∂t** - Legea inducției electromagnetice (Faraday)
4. **∇ × B = μ₀J + μ₀ε₀∂E/∂t** - Legea lui Ampère (extinsă de Maxwell)

### Constante Fizice Utilizate

- **Viteza luminii**: c = 299,792,458 m/s
- **Constanta lui Planck**: h = 6.626 × 10⁻³⁴ J·s
- **Permeabilitatea vidului**: μ₀ = 4π × 10⁻⁷ H/m
- **Permitivitatea vidului**: ε₀ = 8.854 × 10⁻¹² F/m
